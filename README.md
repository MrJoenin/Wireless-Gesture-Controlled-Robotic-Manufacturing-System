# ECET-230-001-Group-Project - Wireless Gesture-Controlled Robotic Manufacturing System
## 🎯 Project Goals
- Build a robotic arm with **5 degrees of freedom (DOF)** controlled by a gyroscope.
- Use an **Arduino Uno** for control.
- Integrate **servo motors** with encoders for precise motion.
- Keep total cost under **$150**.
- Deliver full documentation for replication.
- Include **Ultrasonic** sensor for detection
- include **buzzer** for audio effect
